MemorySavingModeEnum Enumerator

Description

Enum indicating the memory saving mode during drawing view computations.

Methods

Name Value Description
kAlwaysSaveMemoryMode 65282 Memory conservation should be done for all drawing view computations.
kNeverSaveMemoryMode 65283 None of drawing view computations should do memory conservation.
kUseApplicationOptionsMemoryMode 65281 Memory saving mode belong to the application options.

Version

Introduced in version 2008
