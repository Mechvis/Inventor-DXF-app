ShellDirectionEnum Enumerator

Description

Enum specifying the shell boundary relative to the part face.

Methods

Name Value Description
kBothSidesShellDirection 41219 offsets the shell wall equal distances to inside and outside the part.
kInsideShellDirection 41217 Offsets the shell wall to the part interior.
kOutsideShellDirection 41218 Offsets the shell wall to the exterior of the part.

Version

Introduced in version 9
