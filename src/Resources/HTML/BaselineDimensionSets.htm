BaselineDimensionSets Object


Description

The BaselineDimensionSets collection object provides access to all of the baseline dimension sets on the sheet.

Methods

Name Description
Add Method that creates a baseline dimension set and returns the newly created BaselineDimensionSet object.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in the collection.
Item Method that returns the specified dimension set object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

DrawingDimensions.BaselineDimensionSets

Samples

Name Description
Baseline dimension sets This sample demonstrates the creation of a baseline set dimension in a drawing.
