OLEVerbEnum Enumerator

Description

Actions that can be performed on an OLE attachment.

Methods

Name Value Description
kDefaultOLEVerb 3076 Perform the default action associated with this attachment.
kEditOpenOLEVerb 3073 Open the document in a separate window for editing.
kHideOLEVerb 3075 If the document has been opened, hide it. Else, no action.
kShowOLEVerb 3074 If the document has been hidden, display it. Else, open for editing (same as kEditOpenOLEVerb).

Version

Introduced in version 4
