CircularPatternFeatureProxy.Definition Property

Parent Object: CircularPatternFeatureProxy

Description

Read-write property that gets and sets the CircularPatternFeatureDefinition object associated with this feature.

Syntax

CircularPatternFeatureProxy.Definition() As CircularPatternFeatureDefinition

Property Value

This is a read/write property whose value is a CircularPatternFeatureDefinition.

Version

Introduced in version 2017
