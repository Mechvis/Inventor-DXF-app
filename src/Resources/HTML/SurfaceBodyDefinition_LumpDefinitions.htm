SurfaceBodyDefinition.LumpDefinitions Property

Parent Object: SurfaceBodyDefinition

Description

Property that returns the collection of LumpDefinition objects associated with this SurfaceBodyDefinition object. The primary use of the LumpDefinitions collection is to create new LumpDefinition objects.

Syntax

SurfaceBodyDefinition.LumpDefinitions() As LumpDefinitions

Property Value

This is a read only property whose value is a LumpDefinitions.
