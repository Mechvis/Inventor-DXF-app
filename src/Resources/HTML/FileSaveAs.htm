FileSaveAs Object


Description

Custom interface to access management information about a particular file version.

Methods

Name Description
AddFileToSave Adds a document, taken from within this tree, to the set of documents to be saved. Please note: Saving of files in Apprentice is not allowed on files that require migration (any file that has not already been migrated to the same version of the Apprentice server). The current document's NeedsMigrating property must return False before the FileSaveAs object can successfully be returned.
ExecuteSave Executes the Save operation on each of the documents specified via the AddFileToSave method.
ExecuteSaveAs Executes the Save As operation on each of the documents specified via the AddFileToSave method.
ExecuteSaveCopyAs Executes the Save Copy As operation on each of the documents specified via the AddFileToSave method.

Accessed From

ApprenticeServerComponent.FileSaveAs

Version

Introduced in version 4
