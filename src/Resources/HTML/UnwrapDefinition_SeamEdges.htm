UnwrapDefinition.SeamEdges Property

Parent Object: UnwrapDefinition

Description

Read-write property that gets and sets an EdgeCollection that specifies the seam edges.

Syntax

UnwrapDefinition.SeamEdges() As EdgeCollection

Property Value

This is a read/write property whose value is an EdgeCollection.

Version

Introduced in version 2025
