FlatPatternFeatures.SketchDrivenPatternFeatures Property

Parent Object: FlatPatternFeatures

Description

Gets the collection object that besides listing out the subset of part features that are SketchDrivenPatterns, allows the creations of new SketchDrivenPatterns.

Syntax

FlatPatternFeatures.SketchDrivenPatternFeatures() As SketchDrivenPatternFeatures

Property Value

This is a read only property whose value is a SketchDrivenPatternFeatures.

Version

Introduced in version 2017
