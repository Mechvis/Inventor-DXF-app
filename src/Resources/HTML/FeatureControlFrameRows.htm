FeatureControlFrameRows Object


Description

The FeatureControlFrameRows object provides access to all rows of a feature control frame.

Methods

Name Description
Add Method that adds a row to the feature control frame.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in the collection.
Item Method that returns the specified FeatureControlFrameRow object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

FeatureControlFrame.FeatureControlFrameRows, FeatureControlFrames.CreateFeatureControlFrameRows

Samples

Name Description
Adding and editing a feature control frame These samples demonstrate editing an existing feature control frame symbol. The first sample adds a row to an existing symbol. The second sample replaces all rows of an existing symbol.
Create a feature control frame symbol This sample demonstrates the creation of a feature control frame symbol.
