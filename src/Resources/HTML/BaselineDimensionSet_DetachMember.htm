BaselineDimensionSet.DetachMember Method

Parent Object: BaselineDimensionSet

Description

Method that detaches the member from the set. The member is not deleted, it is merely converted into a vanilla linear general dimension.

Syntax

BaselineDimensionSet.DetachMember( Member As LinearGeneralDimension )

Parameters

Name Type Description
Member LinearGeneralDimension

Version

Introduced in version 2010
