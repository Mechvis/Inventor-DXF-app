ModelCompositeAnnotations.Add Method

Parent Object: ModelCompositeAnnotations

Description

Adds a new ModelCompositeAnnotation object.

Syntax

ModelCompositeAnnotations.Add( Definition As ModelCompositeAnnotationDefinition, [ClientId] As Variant ) As ModelCompositeAnnotation

Parameters

Name Type Description
Definition ModelCompositeAnnotationDefinition Input ModelCompositeAnnotationDefinition object that specifies the inputs to create a new ModelCompositeAnnotation.
ClientId Variant Input String value that uniquely identifies the client. This is the CLSID of the AddIn in a string form, e.g. "{C9A6C580-3817-11D0-BE4E-080036E87B02}".

This is an optional argument whose default value is null.

Version

Introduced in version 2018
