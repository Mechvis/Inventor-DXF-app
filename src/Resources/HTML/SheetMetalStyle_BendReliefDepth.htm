SheetMetalStyle.BendReliefDepth Property

Parent Object: SheetMetalStyle

Description

Gets and sets the expression that defines the value of the bend relief depth.

Syntax

SheetMetalStyle.BendReliefDepth() As String

Property Value

This is a read/write property whose value is a String.
