ClientFeatureDefinition.ClientFeatureElements Property

Parent Object: ClientFeatureDefinition

Description

Property that returns the ClientFeatureElements collection object that defines the Inventor objects composited by this feature.

Syntax

ClientFeatureDefinition.ClientFeatureElements() As ClientFeatureElements

Property Value

This is a read only property whose value is a ClientFeatureElements.

Version

Introduced in version 2008
