Point Object


Description

The Point object. The object is a transient mathematical object and is not displayed graphically. For more information see the Transient Geometry article in the overviews section.

Methods

Name Description
Copy Creates a copy of this Point object. The result is entirely independent and can be edited without affecting the original Point object.
DistanceTo Determine the distance between this point and the specified point.
GetPointData Get the data defining this point.
IsEqualTo Compares this point for equality with the specified point.
PutPointData Method that sets the data defining this point.
TransformBy Transform this point by the specified matrix.
TranslateBy Translate this point by the specified vector.
VectorTo Gets the vector of translation between this point and the specified point.

Properties

Name Description
X Read-write proerty that gets and sets the X coordinate value.
Y Read-write proerty that gets and sets the Y coordinate value.
Z Read-write proerty that gets and sets the Z coordinate value.
