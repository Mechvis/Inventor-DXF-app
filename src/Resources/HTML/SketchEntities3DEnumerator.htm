SketchEntities3DEnumerator Object


Description

The SketchEntities3DEnumerator object provides access to a set of 3D sketch entities. See the article in the overviews section.

Properties

Name Description
Count Property that returns the number of items in the collection.
Item Returns the specified SketchEntity3D object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.
