DetailDrawingView.SetComponentLineColor Method

Parent Object: DetailDrawingView

Description

Method that sets color for a component in the drawing view.

Syntax

DetailDrawingView.SetComponentLineColor( Component As Object, Color As Color )

Parameters

Name Type Description
Component Object Input Document, ComponentOccurrence, ComponentOccurrenceProxy, SurfaceBody, SurfaceBodyProxy, PartFeature or PartFeatureProxy object to indicate a component in the drawing view.
Color Color Input Color object to set the color for the component in the drawing view. This the color has its ColorSourceType set to kLayerColorSource or kAutomaticColorSource then the color of the component will be set by layer or automatic.

Version

Introduced in version 2025.1
