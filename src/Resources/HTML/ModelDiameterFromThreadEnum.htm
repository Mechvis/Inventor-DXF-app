ModelDiameterFromThreadEnum Enumerator

Description

Constants identifying the specific diameter out of the thread definition that is to be used for modeling.

Methods

Name Value Description
kThreadMajorDiameter 21761 Diameter of cylindrical face in the model is picked as the Major Thread diameter.
kThreadMinorDiameter 21762 Diameter of cylindrical face in the model is picked as the Minor Thread diameter.
kThreadPitchDiameter 21763 Diameter of cylindrical face in the model is picked as the Pitch diameter of the Thread.
kThreadTapDrillDiameter 21764 Diameter of cylindrical face in the model is picked as the Tap Drill diameter of the Thread.

Version

Introduced in version 5
