LicenseRoleEnum Enumerator

Description

Enumerator that defines license role type.

Methods

Name Value Description
kLicenseRoleExtension 121859 License role extension. The feature/service for non-core functionalities of product.
kLicenseRoleHost 121857 License role host. The feature/service is for core functionalities of product.
kLicenseRolePlugin 121858 License role plugin. The feature/service which can be installed seperately from product.

Introduced in version 2023
