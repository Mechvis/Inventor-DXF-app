PresentationEvents Object


Description

The Presentation Events Object.

Properties

Name Description
Application Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server.
Parent Gets the parent object from whom this object can logically be reached.
Type Gets the constant that indicates the type of this object.

Events

Name Description
OnDelete Event that fires just before and after an entity is deleted.
OnNewPublication Event that fires just before and after a publication is created.
OnNewPublicationMarkedView Event that fires just before and after a publication marked view is added.
OnPublicationComponentChange Event that fires just before and after a publication component is changed.
OnPublicationDesignViewChange Event that fires just before and after a publication design view is changed.
OnPublicationModelingDataUpdate Event that fires just before and after a publication component is changed.
OnPublicationNameChange Event that fires just before and after a publication component’s name is changed.
OnStoryboardChange Event that fires just before and after a publication is updated because of modeling data changes.

Version

Introduced in version 2017
