SurfaceTextureSymbolDefinition Object.

Properties

Name Description
AllAroundSymbol Gets and sets whether to add the all-around indicator to the symbol.
Application Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server.
IsForceTailShown Gets and sets whether to add a tail to the symbol.
IsMajority Gets and sets whether this symbol specifies the standard surface characteristics for the drawing.
Parent Gets the parent object from whom this object can logically be reached.
StandardReference Returns the standard reference type.
SurfaceTextureType Gets and sets the surface type for the symbol.
Type Gets the constant that indicates the type of this object.
