FileManager.GetTemplateFile Method

Parent Object: FileManager

Description

Method that specifies the template to use when creating a file.

Syntax

FileManager.GetTemplateFile( DocumentType As DocumentTypeEnum, [SystemOfMeasure] As SystemOfMeasureEnum, [DraftingStandard] As DraftingStandardEnum, [DocumentSubType] As Variant ) As String

Parameters

Name Type Description
DocumentType DocumentTypeEnum Input constant that specifies the type of document.
SystemOfMeasure SystemOfMeasureEnum Input constant that specifies the system of measure to use in the drawing being created.

If the DocumentType argument is specified to be kDrawingDocumentObject, the method will honor the application option Default Drawing File Type (equivalent to DrawingOptions.DefaultDrawingFileType) to decide whether to use an idw template or an Inventor DWG template.

This is an optional argument whose default value is 8961.
DraftingStandard DraftingStandardEnum Input constant that specifies the drafting standard to use in the drawing being created.



This is an optional argument whose default value is 9729.
DocumentSubType Variant Input constant (GUID) that specifies the subtype of the document to be created (sheet metal or weldment).

This is an optional argument whose default value is null.
