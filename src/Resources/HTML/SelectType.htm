SelectType Enumerator

Description

DMAC list of bit-fields that are used to specify the selection region relative to a shape.

Methods

Name Value Description
SELECTTYPE_INSIDE 1 Select the elements inside the polygon.
SELECTTYPE_OVERLAP 2 Select elements overlapping either INSIDE or OUTSIDE.

Version

Introduced in version 4
