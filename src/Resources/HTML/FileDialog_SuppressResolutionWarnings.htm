FileDialog.SuppressResolutionWarnings Property

Parent Object: FileDialog

Description

Specifies whether or not to warn that a file is being saved outside of a project resolvable location.

Syntax

FileDialog.SuppressResolutionWarnings() As Boolean

Property Value

This is a read/write property whose value is a Boolean.

Version

Introduced in version 11
