PartFeature.AppearanceSourceType Property

Parent Object: PartFeature

Description

Gets and sets the source of the appearance for the feature.

Syntax

PartFeature.AppearanceSourceType() As AppearanceSourceTypeEnum

Property Value

This is a read/write property whose value is an AppearanceSourceTypeEnum.

Samples

Name Description
Removes all appearance overrides in a part. This sample removes all appearance overrides that have been placed on a part.
