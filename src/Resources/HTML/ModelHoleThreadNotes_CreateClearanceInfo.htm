ModelHoleThreadNotes.CreateClearanceInfo Method

Parent Object: ModelHoleThreadNotes

Description

Method that creates a new HoleClearanceInfo object that can be used in editing HoleFeature objects.

Syntax

ModelHoleThreadNotes.CreateClearanceInfo( FastenerStandard As String, FastenerType As String, FastenerSize As String, [FastenerFitType] As Variant ) As HoleClearanceInfo

Parameters

Name Type Description
FastenerStandard String
FastenerType String
FastenerSize String
FastenerFitType Variant

This is an optional argument whose default value is 84072099.

Version
