ExtrudeFeatures.Add Method

Parent Object: ExtrudeFeatures

Description

Method that creates a new Extrude feature.

Syntax

ExtrudeFeatures.Add( Definition As ExtrudeDefinition ) As ExtrudeFeature

Parameters

Name Type Description
Definition ExtrudeDefinition Input ExtrudeDefinition object that defines the extrude feature you want to create. An ExtrudeDefinition object can be created using the ExtrudeFeatures.CreateExtrudeDefinition method. It can also be obtained from an existing ExtrudeFeature object.
