Sketch3DProxy.GeometricConstraints3D Property

Parent Object: Sketch3DProxy

Description

Property that returns a collection of all geometric constraints on the 3D sketch.

Syntax

Sketch3DProxy.GeometricConstraints3D() As GeometricConstraints3D

Property Value

This is a read only property whose value is a GeometricConstraints3D.

Version

Introduced in version 6
