StartupActionTypeEnum Enumerator

Description

Enum indicating the type of startup action to perform each time Autodesk Inventor is opened.

Methods

Name Value Description
kFileNewDialogStartupAction 70913 Cause the File New dialog box to be displayed each time Autodesk Inventor is opened.
kFileOpenDialogStartupAction 70914 Cause the File Open dialog box to be displayed each time Autodesk Inventor is opened.
kNewFileFromTemplateStartupAction 70915 Cause a new file to be automatically created each time Autodesk Inventor is opened.
kNoStartupAction 70916 Cause no startup action to be performed.

Version

Introduced in version 2008
