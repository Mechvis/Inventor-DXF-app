FlatPatternOrientation Object


Description

FlatPatternOrientation Object.

Methods

Name Description
Activate Method that activates the FlatPatternOrientation object.
Copy Method that creates a new FlatPatternOrientation by copying an existing FlatPatternOrientation object.
Delete Method that deletes the FlatPatternOrientation object.

Properties

Name Description
AlignmentAxis Read-write property that gets and sets the alignment axis of flat pattern orientation.
AlignmentRotation Read-only property that returns the parameter object controlling the alignment rotation angle of flat pattern.
AlignmentType Read-write property that gets and sets the alignment type of flat pattern orientation.
Application Gets the root Application object. Where the property is weakly-typed, it can be set to (QueryInterfaced for) 'Application' when running with Inventor whereas, 'ApprenticeServer' when running with the Apprentice Server.
FlipAlignmentAxis Read-write property that gets and sets whether alignment direction of flat pattern orientation is the opposite of the natural direction of the alignment axis.
FlipBaseFace Read-write property that gets and sets whether to flip the flat pattern by swapping the top and bottom faces or not.
Name Read-write property that gets and sets the name for the FlatPatternOrientation object.
Parent Gets the parent object from whom this object can logically be reached.
Type Gets the constant that indicates the type of this object.

Accessed From

FlatPatternOrientation.Copy, FlatPatternOrientations.ActiveFlatPatternOrientation, FlatPatternOrientations.Item

Version

Introduced in version 2015
