RotateTranslateiMateDefinition.AttributeSets Property

Parent Object: RotateTranslateiMateDefinition

Description

Property that returns the AttributeSets collection object associated with this object.

Syntax

RotateTranslateiMateDefinition.AttributeSets() As AttributeSets

Property Value

This is a read only property whose value is an AttributeSets.
