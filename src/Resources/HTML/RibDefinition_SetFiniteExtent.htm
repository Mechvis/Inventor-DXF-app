RibDefinition.SetFiniteExtent Method

Parent Object: RibDefinition

Description

Method that sets the extent type to be a finite distance.

Syntax

RibDefinition.SetFiniteExtent( Distance As Variant )

Parameters

Name Type Description
Distance Variant Input Variant that defines the distance associated with the extent of the rib feature. This can be either a numeric value or a string. A parameter for this value will be created and the supplied string or value is assigned to the parameter. If a value is input, the units are centimeters. If a string is input, the units can be specified as part of the string or it will default to the current length units of the document.

Version

Introduced in version 2012
