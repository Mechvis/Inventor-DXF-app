DataMedium Object


Description

The DataMedium object is used when exporting/importing to various formats using the TranslatorAddIn APIs. See the 'export' samples under TranslatorAddin for examples of its use.

Properties

Name Description
FileName Specifies the name of a data file.
IDataObject Specifies a IDataObject containing the data.
MediumType Specifies the type of the data.
Stream Specifies an IStream containing the data.
String Specifies a String containing the data.

Accessed From

TransientObjects.CreateDataMedium
