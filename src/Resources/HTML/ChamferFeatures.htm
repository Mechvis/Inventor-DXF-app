ChamferFeatures Object


Description

The ChamferFeatures collection object provides access to all of the objects in a component definition and provides methods to create additional ChamferFeature objects.

Methods

Name Description
AddUsingDistance Method that creates a new ChamferFeature defined by a specified distance. The new ChamferFeature is returned.
AddUsingDistanceAndAngle Method that creates a new ChamferFeature that is defined by a distance and at a specified angle to an input face. The new ChamferFeature is returned.
AddUsingTwoDistances Method that creates a new ChamferFeature that is defined by two distances. The new ChamferFeature is returned.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in this collection.
Item Method that returns the specified ChamferFeature object from the collection
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

Features.ChamferFeatures, FlatPatternFeatures.ChamferFeatures, PartFeatures.ChamferFeatures, SheetMetalFeatures.ChamferFeatures

Version

Introduced in version 5
