NonParametricBaseFeatureDefinition Object


Description

The NonParametricBaseFeatureDefinition is a utility object used for creating, querying, and editing non-parametric base features. A NonParametricBaseFeatureDefinition object can be created using the CreateDefinition method of the NonParametricBaseFeatures collection object. They can also be obtained from existing NonParametricBaseFeature objects.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
BRepEntities Read-write property that defines the geometry to be used for creating the non-parametric base feature.
DeleteOriginal Read-write property that defines if the original geometry in the construction environment is deleted.
IsAssociative Read-write property that defines if the copied geometry should be associative to the original geometry.
OutputType Read-write property that specifies the desired result type.
TargetOccurrence Read-write property that specifies the desired target Occurrence.

Accessed From

NonParametricBaseFeatures.CreateDefinition
