ImportedSurfaceOrganizationTypeEnum Enumerator

Description

Specifies the surface organization type.

Methods

Name Value Description
kImportedAsCompositeFeatures 108802 Indicates surfaces are imported as composite features.
kImportedAsSingleCompositeFeature 108803 Indicates surfaces are imported as a single composite feature.
kImportedAsStitchedComposites 108804 Indicates surfaces are imported as auto-stitched composite features.
kImportedAsSurfaceBodies 108801 Indicates the surfaces are imported as individual bodies.

Version

Introduced in version 2016
