SelectSet Object


Description

The SelectSet object acts as a container for all of the objects that are currently selected using the Select command. See the article in the overviews section.

Methods

Name Description
Clear Method that removes all objects from the select set.
Delete Method that deletes all deletable objects in the select set. This is the equivalent of the Delete command. Not all objects that are selectable can be deleted. For example, you can select an edge, but it cannot be deleted. Any undeletable objects in the select set are ignored.
Remove Method that removes the specified object from the select set.
Select Method that adds an entity to the select set.
SelectMultiple Adds multiple entities to the select set.

Properties

Name Description
Count Property that returns the number of objects in the select set.
Item Returns the specified object from the select set.
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

AssemblyDocument.SelectSet, Document.SelectSet, DrawingDocument.SelectSet, PartDocument.SelectSet, PresentationDocument.SelectSet
