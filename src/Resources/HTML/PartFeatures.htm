PartFeatures Object


Description

The PartFeatures collection object. Presents the current view of objects and allows their creation.

Methods

Name Description
CreatePath Method that creates a path used to define the shape of several part features such as sweep, rectangular pattern, split, etc. All other 2D and 3D curves that are connected to the input curve are obtained and used to create a Path object. The new Path is returned.
CreateSpecifiedPath Method that creates a path used to define the shape of several part features such as sweep, rectangular pattern, split, etc. The new path is returned. This method will fail if the input curves are not connected end-to-end.
