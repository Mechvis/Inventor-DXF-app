Edge.Geometry Property

Parent Object: Edge

Description

Property that returns the underlying geometry of the edge (e.g. Arc2D, Circle, Cone etc.)

Syntax

Edge.Geometry() As Object

Property Value

This is a read only property whose value is an Object.
