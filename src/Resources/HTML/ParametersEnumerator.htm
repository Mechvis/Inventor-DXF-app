ParametersEnumerator Object


Description

The ParametersEnumerator object allows enumeration through available parameters.

Properties

Name Description
Count Property that returns the number of items in this collection.
Item Returns the specified Parameter object from the collection. This is the default property of the ParametersEnumerator collection object.
Type Returns an ObjectTypeEnum indicating this object's type.
