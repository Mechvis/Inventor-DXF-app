DrawingBOMs Object


Description

The DrawingBOMs collection object returns the collection of all locally stored BOMs in the drawing document.

Methods

Name Description
IsDrawingBOMDefined Method that returns whether BOM properties have been defined in the drawing for the input model.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in the collection.
Item Returns the specified DrawingBOM object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

DrawingDocument.DrawingBOMs

Version

Introduced in version 2009
