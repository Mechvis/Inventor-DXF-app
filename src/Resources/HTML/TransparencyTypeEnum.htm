TransparencyTypeEnum Enumerator

Description

Constants specifying the transparency quality.

Methods

Name Value Description
kBlendingTransparency 58625 Provides a high quality transparency display that is achieved by averaging the colors of overlapping objects.
kScreenDoorTransparency 58626 Provides a lower quality transparency display that is achieved by using a pattern that allows the color of the hidden object to show through.
