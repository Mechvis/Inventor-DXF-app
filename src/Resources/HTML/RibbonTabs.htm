RibbonTabs Object


Description

The RibbonTabs collection object provides access to all existing tabs within a ribbon and provides methods to create additional tabs.

Methods

Name Description
Add Method that creates a new RibbonTab within a ribbon. The newly created RibbonTab is returned.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in this collection.
Item Returns the specified RibbonTab object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

Ribbon.RibbonTabs
