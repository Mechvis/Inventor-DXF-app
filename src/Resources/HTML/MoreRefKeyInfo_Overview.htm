Exceptional Cases

Profile Editing

You can edit a profile for a feature to include additional intersecting sketch entities so that a face or an edge gets split. In such a case, one of the resulting face/edge will the primary match. However the choice is not deterministic. The other faces/edges will not be returned.

Face Draft/ Face Splits

In some special cases, new faces get generated when you add face drafts or face splits. You could argue that these new faces are also candidates for the purpose of multiple matches. However these new faces will not be returned.
