SketchEllipses Object


Description

The SketchEllipses object provides access to all the objects in a sketch and provides methods to create additional sketch ellipses. See the article in the overviews section.

Methods

Name Description
Add Method that creates a new sketch ellipse.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in this collection.
Item Returns the specified SketchEllipse object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.
