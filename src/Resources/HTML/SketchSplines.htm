SketchSplines Object


Description

The SketchSplines object provides access to all the objects in a sketch and provides methods to create additional sketch splines. See the article in the overviews section.

Methods

Name Description
Add Method that creates a sketch spline through the set of input points.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Count Property that returns the number of items in this collection.
Item Returns the specified SketchSpline object from the collection.
Type Returns an ObjectTypeEnum indicating this object's type.

Accessed From

DrawingSketch.SketchSplines, PlanarSketch.SketchSplines, PlanarSketchProxy.SketchSplines, Sketch.SketchSplines, SketchBlockDefinition.SketchSplines, SketchBlockDefinitionProxy.SketchSplines

Samples

Name Description
Spline - create NURBS This sample demonstrates the creation of a sketch spline using a geometry definition (a NURB). The API also supports creation of 3D sketch splines in a similar way.
