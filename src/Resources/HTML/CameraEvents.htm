CameraEvents Object


Description

The CameraEvents object supports a set of properties and events used to work with a camera object.

Properties

Name Description
Application Returns the top-level parent application object. When used the context of Inventor, an Application object is returned. When used in the context of Apprentice, an ApprenticeServer object is returned.
Parent Gets the parent object from whom this object can logically be reached.
Type Returns an ObjectTypeEnum indicating this object's type.

Events

Name Description
OnCameraChange Event that fires when the camera of a view has been modified but before the view has been updated. You can modify client graphics in response to this event and the modified client graphics will be drawn when the view is updated.

Accessed From

Application.CameraEvents

Version

Introduced in version 2011
