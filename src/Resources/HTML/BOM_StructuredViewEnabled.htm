BOM.StructuredViewEnabled Property

Parent Object: BOM

Description

Gets and sets whether the structured BOM view is enabled.

Syntax

BOM.StructuredViewEnabled() As Boolean

Property Value

This is a read/write property whose value is a Boolean.
