RowMergeTypeEnum Enumerator

Description

Enum indicating the settings for merging hole table rows.

Methods

Name Value Description
kCombineNotesRowMerge 77315 Combine Notes Row Merge combines the Description cells for the holes with the same hole note in the hole table.
kNoRowMerge 77313 No Row Merge preserves legacy hole table formatting.
kRollupRowMerge 77314 Rollup Row Merge combines all rows of the same hole type into one row in the hole table.
