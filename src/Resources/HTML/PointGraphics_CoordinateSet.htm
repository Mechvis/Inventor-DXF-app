PointGraphics.CoordinateSet Property

Parent Object: PointGraphics

Description

Gets and sets the GraphicsCoordinateSet associate with set.

Syntax

PointGraphics.CoordinateSet() As GraphicsCoordinateSet

Property Value

This is a read/write property whose value is a GraphicsCoordinateSet.
